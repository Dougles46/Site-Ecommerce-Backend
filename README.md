
# Sobre o projeto
Projeto de e-commerce desenvolvido em PHP, rodando em ambiente local com XAMPP,
com foco em lógica de backend e fluxo de pedidos.

# Funcionalidades
- Cadastro de produtos
- Carrinho de compras
- Finalização de pedidos
- Área administrativa básica

# Tecnologias
- PHP
- HTML
- CSS
- MySQL
- XAMPP

# Como executar
1. Instalar XAMPP
2. Clonar o repositório
3. Colocar a pasta em `htdocs`
4. Iniciar Apache e MySQL
5. Importar o banco de dados (se houver)
